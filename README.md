## my neovim config

you can directly do download zip and completely replace the config (including lazy-lock.json), lock.json is correct for version8

or 

### installation
1. visit lazyvim, install, rm -rf there .git
2. git clone this in ~/.config/

```
cd ~/.config
git clone git@github.com:kehsihba-etahdiv/nvim-config.git nvim
```

3. install or fix if any broken old rust-analyzer stuff
4. done

5. also install starship prompt ( default )

6. ghostty settings -
```
theme=Github Dark Default
font-size=18
background-opacity=0.8
background-blur=true

keybind = cmd+l=goto_split:previous
```
